Current Role

Built with Angular 18 and TypeScript, the application serves as a data accounting tool to support responsible and transparent data management within government agencies. For testing, we use Spectator, Jest, and Playwright to ensure robustness. By setting clear data guidelines, the project fosters public trust and privacy, aligns with the government's transparency goals, and boosts internal efficiency.
